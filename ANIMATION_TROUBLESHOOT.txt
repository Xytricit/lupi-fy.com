COMMON ERROR MESSAGES:
=====================

ERROR: "Cannot read property 'animations' of undefined"
FIX: AssetManager not initialized
  - Refresh page (Ctrl+F5)
  - Wait for page to fully load before uploading
  - Check network tab (F12 > Network) for failed requests

ERROR: "showToast is not defined"
FIX: Page didn't load completely
  - Wait 3 seconds after page loads
  - Refresh page
  - Check console for other errors

ERROR: "Modal is null"
FIX: Modal HTML didn't create
  - This is a serious error, page likely broken
  - Refresh page (Ctrl+F5)
  - Try accessing editor from fresh tab

ERROR: "Cannot read property 'querySelector'"
FIX: Button click happened but modal is gone
  - Modal element removed before button processed
  - Try clicking button again slowly
  - Or reload and try different file
